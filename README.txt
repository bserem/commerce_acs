INTRODUCTION
------------
The Commerce ACS module is an intergration module for Greek Courier
Company called ACS. This module implements the WSDL connection with
ACS servers in order to apply shipping rates according to weight and
dimensions of oredered product(s).
Submodule :
ACS Address validator
It is a suggestion system that allows the user to select the
correct address.
   
REQUIREMENTS
------------
This module requires the following modules:
 * Commerce
 * Commerce Shipping
 * Rules
